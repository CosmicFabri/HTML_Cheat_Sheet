HTML Cheat Sheet

By Fabrizio Guzmán

Document Outline

• <!DOCTYPE html> Version of HTML. Informs a web browser the document being rendered is of type HTML
• <html> HTML document
• <head> Page information. Metadata container
• <body> Page contents

Layout

• <header> Represents a container for introductory content or a set of navigational links
• <nav> Defines a set of navigation links
• <main> Specifies the main content of a document
• <aside> Defines some content aside from the content it is placed in
• <footer> Defines a footer for a document section

Page Information

• <title> Defines the title of the document. The title is text-only and shown in the tab

Links

• <a href=""> Page link
• <a href="mailto"> Email link
• <a name="name"> Anchor
• <a href="#name"> Link to anchor

Objects

• <object> Defines a container for an external resource (page, picture, media, etc.)

Audio, Video & Others

• <img> Used to embed or import images in a webpage
• <audio> Used to embed sound content in the document
• <video> Used to embed videos on a webpage
• <source> Specifies an inline frame; used to embed another document within the current HTML doc

Document Structure

• <h[1-6]> Defines HTML heading
• <span> Inline container used to mark up a part of a text, or a part of a document
• <p> Defines a paragraph. Browsers automatically add a single blank line before and after it

Lists

• <ol> Ordered list
• <ul> Unordered list
• <dl> Definition list

List Elements

• <li> List item
• <dt> Definition term
• <dd> Term description

Forms

• <form> Defines a form
• <label> Input label
• <select> Drop-down box
• <option> Drop-down options
• <input> Form input
• <textarea> Large text input

Table Definition

• <table> Defines a table
• <tr> Table row
• <td> Table cell
• <th> Header cell

Table Structure

• <thead> Table header
• <tbody> Table body
• <tfoot> Table footer
